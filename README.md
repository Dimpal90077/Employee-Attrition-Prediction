# 👨‍💼 Employee Attrition Prediction

## 📌 Overview

This project uses Machine Learning to predict employee attrition and help HR teams identify employees at risk of leaving the organization.

## 🚀 Features

* Employee Attrition Prediction
* HR Analytics Dashboard
* Interactive Visualizations
* Business Insights & Recommendations
* Streamlit Web Application

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Scikit-Learn
* Streamlit
* Plotly

## 📂 Project Structure

```text
Employee-Attrition-Prediction/
├── analysis.ipynb
├── app.py
├── requirements.txt
├── README.md
├── WA_Fn-UseC_-HR-Employee-Attrition.csv
├── images/dashboard.png
└── model/model.pkl
```

## ▶️ Run the Project

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📊 Key Insights

* Overtime is a major driver of attrition.
* Low job satisfaction increases attrition risk.
* Salary alone does not explain employee turnover.
* HR should focus on employee engagement and retention strategies.
